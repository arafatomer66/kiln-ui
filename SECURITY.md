# Security Policy

## Reporting a vulnerability

If you discover a security vulnerability in Kiln UI, please report it privately by emailing **arafatomer66@gmail.com**. Please do **not** open a public issue.

Include:

- A description of the issue and its impact.
- Steps to reproduce or a minimal proof-of-concept.
- Affected versions if known.

I aim to acknowledge reports within 48 hours and provide a fix or mitigation timeline within 7 days for serious issues.

Thanks for helping keep Kiln UI and its users safe.
